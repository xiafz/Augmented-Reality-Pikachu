Look up cell phone architecture and enable developer mode
Install OpenCV manager corrspondingly at
https://sourceforge.net/projects/opencvlibrary/files/opencv-android/

Code developed in Android Studio and tested on API 19 (Galaxy S4)

Usage:
Take a photo with the defult camera app for a textured surface to track
Open the app and follow the prompt to select the photo you just took
Observe the overlay of rectangle for estimated pose.
